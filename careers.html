Game Developer (Unity/C#) - Detailed Description

Responsibilities

  • Implement gameplay mechanics and systems using Unity and C#
  • Optimize game performance across multiple platforms
  • Collaborate with designers to prototype and iterate on features
  • Write clean, maintainable code following established patterns
  • Debug and resolve technical issues throughout development
  • Participate in code reviews and technical discussions
  • Contribute to technical documentation and knowledge sharing

Nice-to-Haves

  • 2+ years of Unity development experience
  • Experience with multiplayer networking
  • Knowledge of shader programming
  • Mobile game development experience
  • Familiarity with version control (Git)
  • Published game projects or portfolio

Tools & Technologies

  • Unity 2022.3+ and C#
  • Git and collaborative workflows
  • Visual Studio or JetBrains Rider
  • Agile development methodologies

Application Requirements

  • Resume highlighting relevant experience
  • Portfolio or Itch.io page showcasing your work
  • GitHub or code samples demonstrating Unity/C# skills
  • Cover letter explaining your interest in Atlas Games

Technical Artist - Detailed Description

Responsibilities

  • Create and maintain art production pipelines and tools
  • Optimize 3D models, textures, and animations for performance
  • Develop custom shaders and visual effects
  • Bridge communication between art and programming teams
  • Establish technical art standards and documentation
  • Troubleshoot and resolve art-related technical issues
  • Research and implement new rendering techniques

Nice-to-Haves

  • Experience with Unity's HDRP/URP pipelines
  • Proficiency in Maya, Blender, or 3ds Max
  • Shader development (HLSL/ShaderGraph)
  • Python or C# scripting for tool development
  • VFX creation (Particle systems, Niagara, etc.)
  • Performance profiling and optimization experience

Tools & Technologies

  • Unity 2022.3+ and rendering pipelines
  • 3D modeling and animation software
  • Photoshop and texture creation tools
  • Version control systems (Git, Perforce)

Application Requirements

  • Resume with technical art experience
  • Portfolio showcasing tools, shaders, and optimizations
  • Examples of technical problem-solving
  • Cover letter detailing your technical art philosophy

Nice-to-Haves

  • Proficiency with Figma, Sketch, or Adobe XD
  • Understanding of Unity UI systems
  • User research and testing experience
  • Basic front-end development skills (HTML/CSS)
  • Motion graphics and animation skills

Tools & Technologies

  • Figma or similar design tools
  • Adobe Creative Suite
  • Unity UI and Canvas systems
  • Prototyping and user testing tools

Application Requirements

  • Resume highlighting design experience
  • Portfolio showcasing UI/UX design work
  • Case studies demonstrating design process
  • Cover letter explaining your design approach

Our Culture & Values

We believe the best work happens when passionate individuals collaborate in an environment of mutual respect and creative freedom.

Frequently Asked Questions

Are you fully remote?

Yes, we're a remote-first company. Team members work from anywhere with reliable internet. We prefer South African time zones for easier collaboration but are open to other regions.

Do you offer internships?

We occasionally offer project-based internships for students or recent graduates. These are typically 3-6 month engagements focused on specific skills and portfolio development.

How long is your hiring process?

Our typical process takes 2-4 weeks from application to offer. This includes portfolio review, initial interview, skill assessment, and final conversation with the team.

Contractor vs. full-time?

We start most relationships as contract-to-hire arrangements. This allows both sides to ensure a good fit before transitioning to permanent employment.

What if my role isn't listed?

We're always interested in exceptional talent. If you have skills that could benefit our games, reach out with "Other" selected and explain your unique value proposition.

Do you sponsor visas?

Currently we work with contractors and remote employees. We don't sponsor work visas, but remote collaboration across borders is welcome.
